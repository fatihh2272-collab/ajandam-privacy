# Gizlilik Politikası

Ajandam uygulaması kullanıcı verilerini toplamaz, saklamaz veya üçüncü taraflarla paylaşmaz.

Uygulama içerisinde girilen tüm veriler yalnızca cihaz üzerinde (local olarak) saklanır ve hiçbir şekilde dış sunuculara gönderilmez.

Uygulama, kullanıcı deneyimini geliştirmek amacıyla herhangi bir kişisel veri toplamaz.

Herhangi bir sorunuz varsa bizimle iletişime geçebilirsiniz.
